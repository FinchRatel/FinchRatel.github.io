1. 新建

layout 默认为 post， 可以通过修改_config.yml中的 default_layout 来指定默认布局

2. 布局

  • post – 文章
  • page
  • draft – 草稿

3. 文件名称

默认以标题作为文件名称
可以通过 new_post_name 来修改默认文件名称

4. 草稿

草稿默认不会展示在页面中，可以在执行是加上 --draft 参数，或把 render_draft 设为 true 来预览草稿。

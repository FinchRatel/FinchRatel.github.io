主题结构

_config.yml

主题的配置文件，修改时会自动更新，无需重启

languages

语言文件夹

layout

布局文件夹，存放主题的模板文件。
内建Swig模板引擎，可以安装插件来支持EJS、Haml或Jade
根据扩展名来决定使用哪一个模板引擎

scripts

脚本文件夹，启动时会自动载入

source

资源文件夹，除了模板以外的Asset都应该放在这里。
_开头的文件或文件夹会被忽略
如果文件可以渲染，解析后存储到public文件夹，否则直接拷贝到public文件夹

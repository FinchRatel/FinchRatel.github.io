模板

每个主题至少应包含一个index模板

模板 用途 回调
index 首页 -
post 文章 index
page 分页 index
archive 归档 index
category 分类 archive
tag 标签 archive

布局

通过布局让多个模板共享相同的结构
默认使用layout布局，可以在front-matter中指定其他布局，或设置为false关闭布局功能
